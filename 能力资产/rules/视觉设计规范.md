# 规则：视觉设计规范（咪咪文人风）

🎨 这份文档是「{{PROJECT_NAME}}」视觉的灵魂。**做任何 UI 都要遵循这套规范，不要瞎发挥。**

## 主题：咪咪文人风

不是花花绿绿的儿童 APP，不是冰冷的工程报表。
**米白底 + 衬线大字 + 深色对比卡 + 大量留白 = 文人小品文。**

## 配色

| 名字 | 色值 | 用途 |
|---|---|---|
| bg | `#f5f1ea` | 主背景（米白） |
| card | `#ffffff` | 普通卡片背景（纯白） |
| cardSoft | `#fdfaf3` | 二级卡片 / 嵌套块 |
| ink | `#2a2520` | 主文字 / 深色对比卡背景 |
| inkSoft | `#3a3128` | 反色卡的一级亮 |
| muted | `#9b9388` | 次要文字 |
| faint | `#cbc3b6` | 最弱文字 / 占位 |
| line | `#ece5d8` | 边框 / 分隔线 |
| accent | `#8a5a3c` | 强调棕（按钮 / 进度条） |
| green | `#5f7a51` | 完成 / 成功 |
| red | `#b56b5f` | 危险 / 超预算 |
| blue | `#5b7691` | 喝水 / 信息 |
| amber | `#c98a3f` | 提示 |

## 字体

```
英文 / 数字标题：Fraunces（衬线）
英文 / 数字正文：Plus Jakarta Sans（无衬线）
中文标题：Noto Serif SC / Songti SC（衬线）
中文正文：Noto Sans SC / PingFang SC（无衬线）
```

类名：
- `.ff-display` = 衬线（标题用）
- 默认 = 无衬线

## 字号梯度

| 用途 | 字号 | 字重 | 行高 |
|---|---|---|---|
| Hero（如周一）| 3.2rem | 500 | 1 |
| 页头标题 | 2.85rem | 500 | 1 |
| 模块标题 | 1.4rem | 500 | tight |
| stat 数字 | 1.55rem | 500 | none |
| 大数字（喝水）| 2.6rem | 500 | none |
| 正文 | 14px / 0.875rem | 400 | relaxed |
| 次要正文 | 12px | 400 | relaxed |
| eyebrow（小字标签）| 11px | 400 | normal |
| 最小 caption | 9-10px | 400 | normal |

## 字间距（letter-spacing）

| 元素 | 字间距 |
|---|---|
| eyebrow 大写英文 | 0.28em |
| section 大写英文 | 0.2em |
| 普通大写英文 | 0.18em |
| 中文 | 0（不调） |

## 圆角

| 元素 | 圆角 |
|---|---|
| 大卡片 | rounded-3xl（24px） |
| 中卡片 / 嵌套块 | rounded-2xl（16px） |
| 按钮 / pill | rounded-full |
| 输入框 | rounded-2xl |
| 小图标按钮 | rounded-md |

## 间距

- 卡片之间：`space-y-4`（16px）
- 模块 section 之间：`space-y-5`（20px）
- 卡片内 padding：`p-5`（20px）
- 列表项之间：`space-y-2`（8px）

## 三种卡片

### 1. 普通卡 `<Card>`
- 背景：白
- 边框：1px ece5d8
- 圆角：3xl
- 用途：99% 信息块

### 2. 深色对比卡 `<Card dark>`
- 背景：ink
- 文字：bg 米白
- 圆角：3xl
- 用途：每页 1 个，放最重要的 stat 或 hero 信息（绝对不要超过 2 个）

### 3. 软卡 / cardSoft 背景
- 嵌套在普通卡里的二级块
- 比如 stat 单格的背景

## 入场动画

- 页面入场：fadein（0.45s）
- 卡片错开：fadein-1 / fadein-2 / ... / fadein-5（每级 60ms 延迟）
- 完成打卡：pop（0.32s 缩放）
- 弹窗：slideup（从底部 0.32s）

## eyebrow 模式

每个标题上方有小字英文 eyebrow：
```
TODAY · MAY 7      ← eyebrow（11px uppercase 0.28em）
周四              ← 大字标题（2.85rem ff-display）
```

5 个 page 的 eyebrow：
- 今日 → `today`
- 健康 → `health`
- 聊天 → `chat`              ← 2026-05-09 新增（PROP-003）
- 记账 → `ledger`
- 时光 → `moments`
- 我 → `profile`（右上角头像入口）

## 右上角头像入口（PROP-003 / ADR-006，2026-05-09）

每个页面的 `PageHeader` 右侧默认渲染一个永久头像按钮，点击跳 `/profile`。

| 维度 | 值 |
|---|---|
| 位置 | PageHeader 右侧，与标题块顶部对齐（`mt-2 shrink-0`） |
| 形状 | 30×30 圆形（`rounded-full` + `width: 30, height: 30`） |
| 描边 | 1px solid `inkSoft`（`#3a3128`） |
| 背景 | bg `#f5f1ea` |
| 内容 | lucide-react `User` 图标，14×14（`w-3.5 h-3.5`），居中，颜色 `ink` |
| 跳转 | `<Link to="/profile">` |
| 例外 | Profile 页本身传 `rightAction={null}`，避免 self-link |

**实现位置**：`{{APP_REPO_DIR}}/src/shared/components.jsx` → `ProfileAvatarLink` 组件 + `PageHeader` 默认值。

**约束**：
- 不顶状态栏（`pt-2` + `mt-2` 双重保险）
- 跟标题字号对齐（与 2.85rem `<h1>` 顶部基本水平）
- Home 页用了 3.2rem hero 标题，仍用同一 `ProfileAvatarLink` 组件保持一致

## 不要做什么

- ❌ 大色块（撞色 / 渐变 / 霓虹）
- ❌ 卡通图标 / emoji 满屏（emoji 用作点缀，不超过 1 行 1 个）
- ❌ 翻译腔文案（"快速记录你的一笔账目"）— 用「记一笔」就行
- ❌ 大量阴影 / 立体效果
- ❌ 5 列以上的 grid（眼晕）
- ❌ 把所有 Card 都做成 dark 反色（失去对比）
